Community C++ MCP SDK

Build Model Context Protocol clients and servers in C++.

cxxmcp is a production-ready C++17 SDK with full protocol coverage, cross-SDK conformance validation, and support for both 2025-11-25 stable and 2026 draft spec versions.

Get Started API Reference Conformance

SDK-first API

Public headers and package targets are centered on protocol, transport, handler, peer, service, client, server, and sdk layers.

Transport Coverage

Built-in stdio, process stdio, Streamable HTTP, and legacy SSE-compatible paths support client and server integration.

OAuth 2.1

Optional auth layer with PKCE, dynamic client registration, token revocation, and server-side authorization endpoints.
