Community C++ MCP SDK

Build Model Context Protocol clients and servers in C++.

cxxmcp is a production-ready C++17 SDK with full protocol coverage, cross-SDK conformance validation, and support for both 2025-11-25 stable and 2026 draft spec versions.

Get Started Tutorials API Reference Conformance

Full Protocol Coverage

Tool, prompt, resource, sampling, elicitation, tasks, progress, cancellation — all MCP capabilities.

109/110 Server, 447/447 Client

Validated against the official conformance runner. Only failure is an upstream TS SDK issue.

Zero Runtime Dependencies

C++17 public API. HTTP (cpp-httplib) and OpenSSL are opt-in via compile-time flags.

Quick Start

find_package(cxxmcp CONFIG REQUIRED)
target_link_libraries(my_server PRIVATE cxxmcp::server)
#include <cxxmcp/peer.hpp>
#include <cxxmcp/run.hpp>

int main() {
    return mcp::ServerPeer::builder()
        .name("demo-server").version("1.0.0").stdio()
        .tool<mcp::protocol::Json, mcp::protocol::Json>("echo",
            [](const mcp::protocol::Json& input) {
                return mcp::protocol::Json{{"echo", input}};
            })
        .run();
}
