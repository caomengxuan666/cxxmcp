Community C++ MCP SDK Candidate

Build Model Context Protocol clients and servers in C++.

cxxmcp provides a C++17 SDK surface with protocol models, transports, Peer/Service entry points, client and server layers, package targets, and release-gated interoperability checks.

Get Started Open API Reference Release Stability

SDK-first API

Public headers and package targets are centered on protocol, transport, handler, peer, service, client, server, and sdk layers.

Transport Coverage

Built-in stdio, process stdio, Streamable HTTP, and legacy SSE-compatible paths support client and server integration.

OAuth 2.1

Optional auth layer with PKCE, dynamic client registration, token revocation, and server-side authorization endpoints.

CMake Consumption

find_package(cxxmcp CONFIG REQUIRED)
target_link_libraries(my_server PRIVATE cxxmcp::server)
target_link_libraries(my_client PRIVATE cxxmcp::client)
